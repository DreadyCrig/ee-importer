Importer lets you import files by a variety of different data sources:

Local File

You can provide Importer with the absolute file path for the file located on your server that you wish to import.

Remote URL

You can provide Importer with the full website URL for the file you wish to import. Both http:// and https:// are supported. In the event the remote file is protected by HTTP Authentication, you can provide a username and password. Note: Username and Password will be encrypted prior to being stored in the ExpressionEngine database for security reasons.

File via FTP

You can provide Importer with the full FTP login for the file you wish to import. You'll need to provide Host, Username, Password, Port, and Server Path. Note: Username and Password will be encrypted prior to being stored in the ExpressionEngine database for security reasons. You may also perform a test for the FTP/SFTP Connection. Using AJAX, a connection test can be performed and the file will be detected but not downloaded to insure your settings work.

File via SFTP

You can provide Importer with the full SFTP login for the file you wish to import. You'll need to provide Host, Username, Password, Port, and Server Path. Note: Username and Password will be encrypted prior to being stored in the ExpressionEngine database for security reasons. You may also perform a test for the FTP/SFTP Connection. Using AJAX, a connection test can be performed and the file will be detected but not downloaded to insure your settings work.

Manually Upload File

This option means that every time you use this Importer Profile, you will need to manually upload a file from your local computer. To allow manual uploading of JSON files, JSON needs to be added to config/mimes.php file:
