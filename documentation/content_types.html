Importer allows you to import into three different Content Types:

  • Channel Entries
  • Members
  • Playa

Channel Entries

Channel Entries importer accepts data in the form of CSV, JSON, and XML via an uploaded, local, or remote file. More information on the allowed Data Sources and Data Types.

Entry and Expiration Dates. These dates can come in various formats when importing and will be coverted automatically. Please read the strtotime() documentation for more information.

Members

Member importer accepts data in the form of CSV, JSON, and XML via an uploaded, local, or remote file. More information on the allowed Data Sources and Data Types.

Playa

The Playa field data importer accepts data from either the native Relationship custom field type or the Playa custom field type (multiple fields into one).
